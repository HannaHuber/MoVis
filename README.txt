# MoVis Movie Society Visualizer
This is a project for the course Information Visualization at TU Wien.

# Author
Hanna Huber (0 0925230)

# Date 
Jun 16th 2017

# Implementation
This application was developped using JetBrains WebStorm and the javascript library D3.

# Run MoVis
You can start MoVis by opening the following link:
- https://hannahuber.github.io/MoVis/
Alternatively, you can open the file index.html in your browser.
The preferred browser is Chrome.

# Known Issues
Due to the large size of the datasets, some operations may take some time. In the filtering view, filtering movies before averaging increases performance.
